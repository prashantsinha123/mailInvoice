# mailInvoice
Send pdf invoice to your customer with SwiftMailer & Hostgator. Now you can send email even via localhost.This is the best way to create
your dynamic invoice by passing the  value of php variable extracted from database.

### Installation:
The recommended way to install Swiftmailer is via Composer:
```
1). composer require "swiftmailer/swiftmailer:^6.0"
2). Download fpdf17 .
```
Try to put them as given in the repository and you are good to go.
